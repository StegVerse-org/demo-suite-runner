SECTION 2: THE BCAT SIMPLEX AND INVARIANT
================================================================================

2.1 THE STATE SPACE
-------------------

The BCAT simplex is defined by four variables with one constraint:

    g + c + a + t = 1.0
    g, c, a, t >= 0

Where:
    g = governance capacity
    c = capacity for change  
    a = admissibility threshold
    t = time/transition parameter

This is a 3-dimensional simplex (tetrahedron) embedded in 4D space.

2.2 THE INVARIANT
-----------------

The GCAT/BCAT invariant is:

    I(x) = g*c + c*a + a*t + t*g

This can be rewritten using the constraint g+c+a+t=1:

    I(x) = gc + ca + at + tg

The invariant equals zero only when the state collapses to specific edges:
- g-a edge (g>0, a>0, c=t=0): I(x) = 0
- c-t edge (c>0, t>0, g=a=0): I(x) = 0

All other valid states have I(x) > 0.

2.3 ADMISSIBILITY CLASSIFICATION
--------------------------------

For a transition from state x to projected state x':

    ALLOW:   I(x') <= 0     (state on admissible edge)
    DENY:    I(x') > 0      (state in interior, but valid simplex)
    FAIL_CLOSED: invalid simplex or unprojectable state

The admissible region has measure ZERO in the 3-simplex.
Random sampling hits ALLOW with probability ~0.5%.

SECTION 3: THE SCALAR — REALITY SELECTOR
================================================================================

3.1 DEFINITION
--------------

The scalar s is computed from the BCAT state:

    lambda(x) = g * c * a * t
    lambda_max = (0.25)^4 = 1/256

    If I(x) <= 0 (on edge):
        s = 0.0  if on g-a edge (quantum-contracted)
        s = 1.0  if on c-t edge (astrophysical-expanded)
        s = 0.5  if at corner (ambiguous)

    Otherwise:
        s = (lambda/lambda_max) / (I(x)/0.25)
        s = min(1.0, max(0.0, s))

3.2 INTERPRETATION
------------------

The scalar measures gap efficiency — how efficiently the entity closes
the gap between irreversibility and touching reality:

    s ≈ 0:   Quantum-contracted — fast closure, minimal deliberation
    s ≈ 0.5: Classical-critical — maximum gap, maximum deliberation  
    s ≈ 1:   Astrophysical-expanded — fast expansion, minimal contraction

3.3 THE SCALAR AS TIME
----------------------

Time is not a separate dimension. Time IS the gap:

    gap_size = information_capacity / rate_of_state_change

    c_eff(s) = c_0 * |s - 0.5|

    At s = 0.5: c_eff = 0 → gap doesn't close → time stops
    At s = 0 or 1: c_eff = c_0 → gap closes at maximum rate → time flows

The scalar is the efficiency asymmetry of the gap:

    s = 0.5 * (1 + efficiency_asymmetry)

SECTION 4: THE GAP — ACTIVE WORKSPACE
================================================================================

4.1 STRUCTURE
--------------

The gap between irreversibility and touching reality has three layers:

    Preparation layer:    What information is assembled
    Negotiation layer:    How coherence is traded against obscurity
    Commitment layer:     What pattern is locked in at contact

4.2 GAP VARIABLES
-----------------

    coherence_budget = fidelity(state_before, projected_state)
                     = sum(g_i * g_i' + c_i * c_i' + a_i * a_i' + t_i * t_i')

    obscurity_gradient = I(x') - I(x)  (invariant change)

    negotiation_depth = number of reversible transitions before commitment

    commitment_latency = time between last reversible transition and contact

4.3 ENTROPY DEBT
----------------

    entropy_debt = H(projected) - H(state_before)

    Where H is Shannon entropy: H(x) = -sum(p_i * log2(p_i))

    The entropy debt is the information lost to obscurity in the gap.
    It must be "paid" at contact through imprint mass-energy:

    E_imprint = k * T * entropy_debt
    m_imprint = E_imprint / c^2

================================================================================

5.1 ENERGY COMPONENTS FROM TRANSITIONS
----------------------------------------

    Ordinary matter:  ALLOW transitions (information successfully condensed)
    Dark matter:      DENY transitions (information left gravitational trace)

5.2 INFORMATION-THEORETIC MASS
------------------------------

    mass ~ shannon_information(state) * scale_factor

    scale_factor = log(scale / l_Planck) / log(l_Hubble / l_Planck)

    scale(s) = l_Planck * exp(alpha * |s - 0.5| * log(l_Hubble/l_Planck))

5.3 SCALAR-PARTITIONED ENERGY BUDGET
------------------------------------

For DENY transitions, partition by scalar regime:

    s < 0.5 (quantum): gravitational trace → dark matter
    s > 0.5 (astrophysical): expansion → dark energy

5.4 ENTITY-WEIGHTED INTEGRAL
----------------------------

The correct cosmological integral weights by entity density and time:

    Omega_i = integral_0^1 [entity_density(s) * time_fraction(s) * component_i(s)] ds

Where:
    entity_density(s) = number of entities with gap efficiency s
    time_fraction(s) = fraction of cosmic time spent at s
    component_i(s) = energy fraction for component i at s

================================================================================

6.1 DEFINITION
--------------

Life is an entity whose scalar oscillates within the viable band:

    0.3 < s_life(t) < 0.7

    s_life(t) = 0.5 + A * sin(omega * t + phi)

Where:
    A = amplitude of deliberation (option exploration)
    omega = frequency of commitment (action rate)
    phi = phase offset (timing relative to stimuli)

6.2 ENTITY CLASSIFICATION
-------------------------

    Rock:       A = 0, omega = 0          → s = constant → not life

---------------------------

    Cost_life = integral_{birth}^{death} [c_eff(s(t)) * I(x(t)) * entropy_debt(t)] dt

This is the total information processed during the entity's lifetime,
weighted by gap closure efficiency.

SECTION 10: THE CIRCLE CONDITION (BLACK HOLE LIMIT)
================================================================================

10.1 GEOMETRY
-------------

When all pairwise relationships between vertices have equal obscurity,
the boundary becomes a circle (or sphere in higher dimensions):

    I_circle(x) = sum of all six pairwise products - k

    At uniform state (g=c=a=t=0.25):
    sum of pairs = 6 * (0.25)^2 = 0.375
    For I = 0: k = 0.375

The admissible region collapses to a single point at the center.

10.2 SCALAR BEHAVIOR
--------------------

Under circle condition:
    s = 0.5 everywhere
    c_eff = 0
    Time stops
    Information is maximally mixed
    Observer cannot distinguish self from system

This is the black hole: maximum symmetry, maximum obscurity,
minimum knowability.

10.3 INFORMATION PARADOX RESOLUTION
-------------------------------------

The reflection A_n recovers coherence from the missing part:

    A + (missing part negotiates with expanded part) → A_n

The missing part becomes obscurity.
The expanded part is coherence borrowed from the future.

The black hole evaporates when the circle breaks — when symmetry
is lost and the scalar becomes informative again.
